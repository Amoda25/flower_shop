I designed and developed Bloomi, a modern and visually appealing flower shop website using HTML and CSS, with a strong focus on simplicity, elegance, and user experience. The project demonstrates my ability to transform a creative concept into a functional web interface by using semantic HTML structure and clean CSS styling. I aimed to create a smooth and enjoyable browsing experience, with intuitive navigation, well-organized sections, and a design that reflects the soft and aesthetic feel of a floral brand.

Through this project, I strengthened my foundation in front-end development by working on responsive layouts, visual design elements, and user-friendly interfaces. Bloomi highlights my attention to detail, creativity, and growing skills in UI design, while showcasing my ability to build clean and structured websites. This project represents an important step in my journey as a developer, combining both technical knowledge and design thinking to create meaningful digital experiences.
